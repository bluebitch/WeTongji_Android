# 命名规范（待补充）

* string.xml

> 比如一个dialog的message

> `msg_dialog_xxxx`（string类型\_所使用的控件名\_具体名字）

> 类型名和控件名最好分别控制在4和6个字符之内

* 控件

> 比如一个login的button

> `btnLogin`(控件类型+用途)

> 类型名和用途最好分别控制在4和6个字符之内

* 其他变量

> 长度控制下，不用太长
> 建议参考这个网页上的规范(http://java.sun.com/docs/codeconv/html/CodeConvTOC.doc.html)
